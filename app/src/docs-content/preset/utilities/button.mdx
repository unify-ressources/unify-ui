### Introduction

The Button component in Preset UI offers a variety of customizable variants for colors and sizes, providing flexibility to match your design requirements.

## Base

Button comes with a base classes that set shared rules for all buttons.

Just add `btn` utility class.

<BlockCode>
```html
<button class="btn">
    Click Me
</button>
```
</BlockCode>

## Variants

Apply a solid, soft, ghost or Outline style to a button

| ClassName | Description | Example |
|-----------|-------------|---------|
| `btn-variantName` | Base class for all buttons with variantName style | `btn-solid` |
| `btn-variantName-colorName` | set colors for the button | `btn-solid-primary` |


## Size 

Set size for a button

| ClassName | Description | Example |
|-----------|-------------|---------|
| `btn-sizeVariant` | `xs`, `sm` ,`md`, `lg`,`xl` | `btn-md` |
| `btn-icon-sizeVariant` | `xs`, `sm` ,`md`, `lg`,`xl` | `btn-icon-lg` |



## Example

<BlockCode>
```html
<button class="btn btn-md btn-soft btn-soft-warning">
    Click Me
</button>
```
</BlockCode>


## Customization

All button variants and sizes are customizable via CSS variables. 

### Solid Variantes

Define a solid style like this 

<BlockCode>
```html
<button class="btn btn-md btn-solid btn-solid-primary">
    Click Me
</button>
```
</BlockCode>

- The `btn-solid` class is defined by the preset
- But for you need to define `btn-solid-primary` class like this :

<BlockCode>
```css
.btn-solid-primary{
    --btn-solid-bg: hsl(var(--primary-600));
    --btn-solid-bg-hover: hsl(var(--primary-700));
    --btn-solid-bg-active: hsl(var(--primary-800));
    --btn-solid-shadow-a: hsl(var(--primary-700));
    --btn-solid-shadow-b: hsl(var(--primary-500));
    --btn-solid-shadow-c: hsl(var(--primary-400));
    --btn-solid-shadow-active-a: hsl(var(--primary-900));
    --btn-solid-shadow-active-b: hsl(var(--primary-700));
    --btn-solid-shadow-active-c: hsl(var(--primary-600));
}
```
</BlockCode>


### Outline

<BlockCode>
```html
<button class="btn btn-md btn-outline btn-outline-primary">
    Click Me
</button>
```
</BlockCode>


<BlockCode>
```css
.btn-outline-primary{
    --btn-outline-color: hsl(var(--primary-500));
    --btn-outline-color-hover: hsl(var(--primary-700));
    --btn-outline-text-color: hsl(var(--primary-600));
    --btn-outline-text-color-hover: hsl(var(--primary-800));
    --btn-focus-outline-color: var(--btn-outline-text-color-hover)
}
```
</BlockCode>


### Ghost

<BlockCode>
```html
<button class="btn btn-md btn-ghost btn-ghost-primary">
    Click Me
</button>
```
</BlockCode>

<BlockCode>
```css
.btn-soft-primary{
    --btn-ghost-bg-color-hover: hsl(var(--primary-200) / .7);
    --btn-ghost-bg-color-press: hsl(var(--primary-200) / .4);
    --btn-ghost-text-color: hsl(var(--primary-700));
}
```
</BlockCode>
