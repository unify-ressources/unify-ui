## Progressbar

Easily customize the semantic `progressbar` element.


<BlockCode>
```html
<progress max="14" value="4" class="progress-bar progress-bar-bg-gray progress-bar-rounded-lg progress-bar-xs text-success600"></progress>
```
</BlockCode> 

- **Set base rules :** Simply add `progress-bar` 
- **Progressebar background** : add `progress-bar-bg-*`
    - `gray` : apply a background with gray color (light gray-200 and dark gray-900)
    - `light` : apply a background with gray color (light gray-100 and dark gray-900)
    - `high` : apply a background with gray color (light gray-200 and dark gray-800)
    - `higher` : apply a background with gray color (light gray-300 and dark gray-700)
- **Progressbar Height** :  add `progress-bar-sizeVariant`
    - `xs`
    - `sm`
    - `md`
    - `lg`
    - `xl`
    - `2xl`
    - `3xl`
- **Corner radius** : add `progress-bar-rounded-*`
    - `sm`
    - `md`
    - `lg`
    - `xl`
    - `full`
